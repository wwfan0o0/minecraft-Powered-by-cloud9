①安装node.js                              下载地址  http://nodejs.org/download/
②安装CoffeeScript （查找“Download zip”）下载地址  https://github.com/jashkenas/coffeescript    
把刚才下载的CoffeeScript的ZIP包，解压到C:\Users\username\AppData\Roaming\npm\node_modules\coffee-script目录中，
记得是coffee-script目录中，注意名称是“coffee- script”
③在node.js command prompt 中键入 coffee connect.coffee snobestar-c9-wwfan.c9.io
之后可以直接链接minecraft
